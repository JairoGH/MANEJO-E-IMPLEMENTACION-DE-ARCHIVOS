# MANEJO-E-IMPLEMENTACION-DE-ARCHIVOS

## Descripción
📂 Proyecto MIA - GoDisk EXT2 Este proyecto implementa un **sistema de archivos basado en EXT2** utilizando el lenguaje de programación **Go**. El objetivo principal es **aprender y comprender** cómo funcionan internamente los discos, particiones, inodos, bl...

- Go
- Go modules
- HTML
